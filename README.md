# Physical AI & Humanoid Robotics Book

## Getting Started

To run the documentation site locally:

1. Clone the repository
2. Navigate to this directory (`001-book-spec`)
3. Install dependencies: `npm install`
4. Start the development server: `npm run start`
5. Open [http://localhost:3000](http://localhost:3000) in your browser

## Learning Path

The book is designed with progressive complexity in mind. We recommend starting with Chapter 1 and working through the lessons sequentially before moving to the projects. Each lesson includes exercises and examples to reinforce learning.

## About This Book

This book emphasizes hands-on learning with practical exercises and projects. It connects directly to real-world applications in AI and robotics, making concepts accessible to beginners while providing depth for intermediate learners. All tutorials, projects, and code snippets are thoroughly documented to ensure comprehensive learning support.
